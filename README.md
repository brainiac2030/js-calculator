JavaScript Calculator
Description
This project is a simple calculator built using JavaScript. It can perform basic operations like addition, subtraction, multiplication, and division.

One of the main features is that it keeps track of all calculations in a history array, which can be displayed when needed.

The goal of this project was to practice using functions, arrays, and objects in JavaScript.
Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers
- Stores all calculations
- Displays calculation history

How it Works
Each operation is written as a separate function. When a calculation is performed, the result is stored in an array as an object.
